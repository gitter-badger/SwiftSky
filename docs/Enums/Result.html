Result

public enum Result

Represents the success or failure of the retrieval/processing of a Forecast object from the Dark Sky API.

Possible Implementations

A Switch Statement (cleanest)

switch result {
case .success(let forecast):
   // do something with the `Forecast`
case .error(let error):
   // do something with `Error`
}

A Conditional Statement

if let forecast = result.response as? Forecast {
   // do something with `Forecast`
} else if let error = result.error as? ApiError {
   // do something with `Error`
}
